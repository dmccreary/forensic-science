# Generate Cover Image Prompt

The following was given to Codex ChatGPT 5.5

!!! prompt
    Please generate the cover images for each of the stories in @docs/stories/*/index.md. You will find the description of the cover image in index.md

    <details>
    <summary>Cover Image Prompt</summary>
    ...

    Once you generate the image, save it into the cover.png file so that this works:

    ```markdown
    ![Cover Image](./cover.png)
    ```

I had had a OpenAI "Plus" plan for $20/month and it used about 30% of the five-hour token limit to generate these images.
