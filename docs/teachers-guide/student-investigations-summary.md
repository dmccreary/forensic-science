# Student Investigations — Summary Table

This is the teacher's planning view of every [Student Investigation](../student-investigations/index.md).
Click any column header to **sort** (click again to reverse). Use it to build a
term plan around your budget, your available lab space, and your safety comfort
level.

- **Type** — *Hands-on Physical* (a real bench lab), *Pure Virtual* (stories +
  MicroSims, no consumables), or *Hybrid* (a physical step plus a virtual step
  that simulates an expensive instrument).
- **Cost** — approximate consumable cost **per group** (shared equipment such as
  microscopes and laptops is assumed on hand). "Free" means no consumables.
- **Risk** — the chance that a **student or lab equipment could be harmed**, rated
  **Low / Med / High**. See the [rating basis](#how-risk-is-rated) below.

**28 investigations:** 9 Hands-on Physical · 8 Pure Virtual · 11 Hybrid.

## How Risk Is Rated

The risk rating reflects the chance that a **student or lab equipment could be
harmed** during the student-performed portion of the lab, assuming normal
classroom PPE and supervision.

| Rating | Meaning | Examples in this book |
|--------|---------|-----------------------|
| <span class="si-risk si-risk-low">Low</span> | No hazardous chemicals, flame, or sharps in the student steps. Everyday or simulated materials. | Tape lifts, chromatography, simulated blood typing, all Pure Virtual labs. |
| <span class="si-risk si-risk-med">Med</span> | Mild chemical irritants or sharp materials that need goggles, gloves, and close supervision. | Kastle-Meyer (hydrogen peroxide), color spot tests (iodine), handling glass chips. |
| <span class="si-risk si-risk-high">High</span> | Open flame and/or flammable solvent — requires a fume hood or strong ventilation and one-station-at-a-time supervision. | Fiber Under Fire (burn test + acetone). |

!!! warning "Always pair with your district's lab-safety policy"
    These ratings are a planning aid, not a substitute for your own risk
    assessment. No investigation uses real blood, real controlled substances, or
    unsupervised flame — every "blood," "drug," and "accelerant" in the book is a
    safe simulant. Confirm PPE, ventilation, and disposal against your local
    requirements before running any physical lab.

<script>
(function () {
  var table = document.getElementById('siTable');
  if (!table || !table.tHead) return;
  var headers = table.tHead.rows[0].cells;
  var tbody = table.tBodies[0];
  var current = { col: -1, dir: 1 };

  function value(row, i, type) {
    var cell = row.cells[i];
    var v = cell.getAttribute('data-sort');
    if (v === null) v = cell.textContent.trim();
    if (type === 'num') { var n = parseFloat(v); return isNaN(n) ? 0 : n; }
    return v.toLowerCase();
  }

  function sortBy(i) {
    var th = headers[i];
    var type = th.getAttribute('data-type') || 'text';
    current.dir = (current.col === i) ? -current.dir : 1;
    current.col = i;
    var rows = Array.prototype.slice.call(tbody.rows);
    rows.sort(function (a, b) {
      var va = value(a, i, type), vb = value(b, i, type);
      if (va < vb) return -current.dir;
      if (va > vb) return current.dir;
      return 0;
    });
    rows.forEach(function (r) { tbody.appendChild(r); });
    for (var h = 0; h < headers.length; h++) {
      if (!headers[h].classList.contains('si-sortable')) continue;
      headers[h].setAttribute('aria-sort',
        h === i ? (current.dir === 1 ? 'ascending' : 'descending') : 'none');
      var caret = headers[h].querySelector('.si-caret');
      if (caret) caret.textContent = (h === i) ? (current.dir === 1 ? '▲' : '▼') : '⇅';
    }
  }

  for (var i = 0; i < headers.length; i++) {
    if (!headers[i].classList.contains('si-sortable')) continue;
    (function (idx) {
      var th = headers[idx];
      th.setAttribute('role', 'button');
      th.setAttribute('tabindex', '0');
      th.addEventListener('click', function () { sortBy(idx); });
      th.addEventListener('keydown', function (e) {
        if (e.key === 'Enter' || e.key === ' ') { e.preventDefault(); sortBy(idx); }
      });
    })(i);
  }
})();
</script>
