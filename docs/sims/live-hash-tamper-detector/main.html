Live Hash & Tamper Detector

Type evidence text and watch its MD5 and SHA-256 fingerprints appear. Flip a single character to see the avalanche effect, then compare two files to authenticate the untouched original.

This tool needs a secure context (open it from http://localhost or file://). SHA-256 hashing is unavailable here.

1 · Evidence text

MD5 (128-bit · 32 hex digits)
SHA-256 (256-bit · 64 hex digits)

2 · Compare two files

Paste the contents of two files. Identical SHA-256 hashes prove they are bit-for-bit the same file — an untouched copy. Any difference means one was tampered with.
